Теги : что это такое и как это применять

Что такое тег (tag)?

Строительный кирпичик сайта. Из тегов состоят все страницы в интернете, такие как Youtube, TikTok, Instagram, Google!

Самые популярные теги:

  • <a> - ссылка
  • <img> - картинка
  • <p> - текст
  • <h1-6> - заголовок
  • <div> - коробка / контейнер
  • <input> - поле ввода данных
  • <ul> - список
  • <li> - элемент списка
  • <button> - кнопка
  • <span> - выделитель части контента
  • <header> - шапка сайта (верхняя часть странички)
  • <nav> - навигация (ссылки-меню странички)
  • <main> - контент сайта (главная часть странички)
  • <footer> - подвал сайта (нижняя часть странички)
  • <link> - подключает файлы к html страничке
  • <script> - тут пишут JavaScript или подключают файл со скриптами

Теги бывают парные и одинарные. В парных тегах подразумевается, что есть начало контента и его конец, а одинарных такого нет.
Например парные:

  • <a>
  • <div>
  • <p>
  • <button>

Одинарные:

  • <img>
  • <input>
  • <br>
  • <hr>

    <button>

    <button> - создаёт кликабельную кнопку, которая может быть использована в любом другом месте документа, который требует простой, стандартной кнопки. .

    <span>

    Он может использоваться для группировки элементов в целях стилизации

    Является основным строковым контейнером для текстового контента, который, по существу, ничего не представляет.

    <span> очень похож на элемент <div>,но <div> является блочным элементом, в то время как span является строчным.

    Блочные элементы занимают всю ширину своего родителя (контейнера), формально создавая «блок» (отсюда и название). строчный элемент это информация (контент). то есть если портфель это дконтейнер div то span это тетрадь с информацией.

    <footer>

    (футер, подвал)

    Футер обычно содержит информацию об авторе раздела, информацию об авторском праве или ссылки на связанные документы. такойже находится внизу страници. место для выжных ссылок для сайта
